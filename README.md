# nxp_ivesliu_skill

This repository is the lightweight, Git-managed view of the local `NXP_v2`
workspace.

It intentionally tracks only:

- workspace entry docs
- local skill definitions
- support-layer entry docs
- board knowledge docs
- source-module `USAGE.md` files

It intentionally does not track heavy local assets such as:

- image bundles
- SDK archives
- toolchains
- source trees
- firmware blobs
- case outputs

Those heavy assets remain in the local workspace and are managed outside this
Git repository.
