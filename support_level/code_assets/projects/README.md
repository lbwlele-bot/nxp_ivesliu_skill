# projects

这里放标准源码资产。

这层只回答三件事：

- 本地有哪些源码项目
- 它们真实放在哪里
- 它们能不能当成标准源码基线来读

使用规则：

- 默认只读
- 可以浏览、比对、切 branch / tag / commit
- 不在这里直接做 case 构建
- 一旦问题变成“我要编什么”，回 `compile` 判断编译对象
