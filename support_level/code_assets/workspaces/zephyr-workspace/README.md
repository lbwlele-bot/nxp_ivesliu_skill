# zephyr-workspace

这是当前本地默认的 Zephyr 工作区入口。

当前保留它，不是因为它必须绑定某个 `RTE` 版本，
而是因为它作为一个整体仍然有独立意义：

- 里面不只是一个源码仓库
- 它包含 `zephyr/` 和配套 `modules/`
- 某些 Zephyr / A55 RTOS 相关任务，需要按这个整体去理解或复用

历史上这份工作区可能来自某一条 `RTE` 线，
但目录名不再把这种历史来源当成当前 owner。

使用规则：

- 先确认当前任务是否确实落到这套 Zephyr 工作区
- 如果只是查源码资产，优先回到 `../../projects/`
- 如果任务已经变成“要编 Zephyr”，优先回 `../../../compile_targets/zephyr/`
- 如果要修改、构建、生成产物，复制到 `../../../work/<case>/` 再做
