# m_freertos_sdk

这里放的是 `NXP MCUX SDK / FreeRTOS SDK` 发布压缩包资产。

它和 `source_code/` 不一样：

- 这里默认是厂商发布包
- 不是普通 Git clone 基线
- 不应被当成“缺了就自己去网上拉”的源码树

使用规则：

1. 先看本地 `m_freertos_sdk/` 里有没有目标板、目标版本的 SDK 压缩包
2. 如果没有，不要自行去网上下载
3. 先找用户要，或者让用户明确提供下载好的发布包
4. 解压、修改、编译，放到当前 `work/<case>/` 下做，不要在这里原地改

当前目录适合长期保留的内容：

- 原始 `.zip` / `.tar.gz` 发布包
- 必要时可附一个很短的版本说明

当前目录不适合长期保留的内容：

- 解压后被改动的工程
- 编译输出
- 临时 patch
- case 级日志
