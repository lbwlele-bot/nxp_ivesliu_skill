# linux_document

这里预留给长期保留的 Linux BSP 官方 docs 包，例如：

- release note
- User's Guide
- build guide
- porting guide

当前状态：

- 旧长期资产根里没有发现已经单独整理好的 Linux docs 包
- 现有像文档的内容大多散落在旧 `work/` case、SDK 压缩包或 Android drop 里
- 为避免把 case 噪声混进长期支撑层，这里暂时不做整包迁入

后续规则：

- 如果后面定位到正式、可复用、版本清晰的 docs 包，再迁进这里
- 不把 case 临时目录里的零散文档直接当成长期 `linux_document` 资产
